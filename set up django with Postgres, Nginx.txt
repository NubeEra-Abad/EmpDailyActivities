* set up django with Postgres, Nginx, and Gunicorn on ubuntu

$ sudo apt-get update


*  installing python3 and postgresql and nginx

$ sudo apt-get install python3-pip python3-dev libpq-dev postgresql postgresql-contrib nginx


* Create the PostgreSQL Database and User

$ sudo -u postgres psql
$ CREATE DATABASE myproject;
$ CREATE USER myprojectuser WITH PASSWORD 'vishal';
$ ALTER ROLE myprojectuser SET client_encoding TO 'utf8';
$ ALTER ROLE myprojectuser SET default_transaction_isolation TO 'read committed';
$ ALTER ROLE myprojectuser SET timezone TO 'UTC';
$ GRANT ALL PRIVILEGES ON DATABASE myproject TO myprojectuser;


* Create a Python Virtual Environment for your Project

$ sudo -H pip3 install --upgrade pip
$ sudo -H pip3 install virtualenv

$ mkdir ~/myproject
$ cd ~/myproject
$ virtualenv myprojectenv
$ source myprojectenv/bin/activate
$ pip install django gunicorn psycopg2


* Create and Configure a New Django Project

$ django-admin.py startproject myproject ~/myproject

* Adjust the Project Settings

$ nano ~/myproject/myproject/settings.py

	ALLOWED_HOSTS ['.vishal.com', '13.233.172.40']



DATABASES = {
    'default': {
        'ENGINE': 'django.db.backends.postgresql_psycopg2',
        'NAME': 'myproject',
        'USER': 'myprojectuser',
        'PASSWORD': 'password',
        'HOST': 'localhost',
        'PORT': '',
    }
}


STATIC_URL = '/static/'
STATIC_ROOT = os.path.join(BASE_DIR, 'static/')


* Complete Initial Project Setup

$ pip install django
$ 	~/myproject/manage.py makemigrations
$ ~/myproject/manage.py migrate
$ ~/myproject/manage.py createsuperuser
$ ~/myproject/manage.py collectstatic

$ sudo ufw allow 8000
$ ~/myproject/manage.py runserver 0.0.0.0:8000
